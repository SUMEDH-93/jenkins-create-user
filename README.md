# 🚀 Create a Jenkins User

## 🎯 Objective

Create a new user in Jenkins and verify successful user creation.

---

## 📋 Prerequisites

- Jenkins Installed and Running
- Administrator Access to Jenkins

---

## 🛠️ Steps

### Step 1: Open Security Settings

Navigate to:

```text
Manage Jenkins → Security
```

### Step 2: Open Users Section

Click:

```text
Users
```

### Step 3: Create New User

Click:

```text
Create User
```

### Step 4: Enter User Details

Fill in the following information:

```text
Username: sumedh
Password: ********
Confirm Password: ********
Full Name: Sumedh
Email Address: sumedh@example.com
```

### Step 5: Create User

Click:

```text
Create User
```

### Step 6: Verify User Creation

Navigate to:

```text
Manage Jenkins → Security → Users
```

Verify that **sumedh** appears in the users list.

---

---

## ✅ Verification

- Open **Manage Jenkins → Security → Users**
- Search for **sumedh**

---

## 🎉 Conclusion

The Jenkins user **sumedh** has been successfully created and verified.
